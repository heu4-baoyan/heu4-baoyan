# 关于

[清华大学飞跃手册](https://feiyue.online)（简称“手册”）是由一群清华大学出国（境）深造的毕业生编写维护的在线文档。

该手册旨在总结出国（境）深造的经验教训，回答申请中遇到的问题，为拟出国（境）深造的学弟学妹提供借鉴和参考，减少信息差带来的不公平，降低准备时间和金钱成本，缓解申请过程中的焦虑。

希望本手册能够帮助到拟出国（境）深造的清华学生，在申请路上顺顺利利，获得自己心仪项目的 Offer！
